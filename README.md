## MIPS Processor Simulator


####git commands:
  1. create a new local branch
  ..- git branch NAME
  2. checkout a branch
  ..- git checkout NAME
  3. create remote branch
  ..- git push -u origin NAME
  4. merge branch to master
  ..- git checkout master
  ..- git merge NAME
  5. delete local branch
  ..- git branch -d NAME
  6. delete remote branch
  ..- git push origin -d NAME
